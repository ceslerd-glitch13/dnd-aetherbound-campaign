## Quick Reference

**How to Use This File**:
- Before a session: Review recent decisions and active quests.
- When a player talks to an NPC: Check this file + **NPC_Backstory_Personality_file.md** for attitude and quest info.
- After a session: Update this file with new decisions and impacts.
- For long-term memory: The summaries section helps you quickly recall what happened without re-reading everything.
