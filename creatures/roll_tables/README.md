# Random Encounter Roll Tables

Use this folder whenever a random encounter roll is requested.

## Required order

1. Open [`Roll_Table_Index.md`](Roll_Table_Index.md).
2. Select the appropriate d6 table. For broad regional requests, use [`Roll_Table_Selection_Guide.md`](Roll_Table_Selection_Guide.md).
3. Roll on the table before consulting individual creature entries.
4. Follow the links in the rolled result's **Catalog follow-up** column.
5. Use the linked creature catalog entries for CR, recommended level, regional placement, source, dead-zone behavior, strengths, and weaknesses.

The roll tables generate encounter groups. They do not replace the creature catalog, complete official stat blocks, the World Bible, Geography, or campaign-specific combat statistics.
