# Token and Image Guidance

> **Repository role:** Supporting asset guidance for the creature catalog. Token approval is separate from creature approval and encounter design.

## Track for each asset

- official image availability;
- existing campaign token;
- whether a new token is required;
- portrait or thumbnail needs;
- token size and encounter role;
- repository asset path;
- usage rights or source restrictions.

## Repository placement

Use `assets/tokens/creatures/` for approved creature tokens when assets are later added. Organize additional subfolders only when the collection becomes large enough to require them.

Recommended filename:

`creature-name__version__role__size.ext`

Dead-zone suppression alone does not change ordinary wildlife's appearance. Separate art is useful only when the physical appearance materially changes, such as an escaped specimen with restraints, a visibly damaged construct, or an approved altered variant.
